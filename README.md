ptz-core-domain is the core for all Polutz modules.


## Use

### Install
```
    npm install --save ptz-core-domain
```

### How to use
```javascript    
    import {
        validateEmail,
        EntityBase,
        EntityMinBase,
        IAppFuncArgs,
        ICreatedBy,
        IEntityMinBase,
        IEntityMinBaseArgs,
        IEntityBase,
        IEntityBaseArgs,
        IBaseRepository,
        IBaseApp,
        IBaseAppArgs
    } from 'ptz-core-domain';
```
